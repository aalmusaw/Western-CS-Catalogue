Welcome to Western's Computer Science Course Catalogue

This is a course project for CS3319A Assignment 3. All the data used is completely fake and the purpose of this project is to practice creating a website that uses a database at the backend and retrieve and edit data using a GUI on the frontend.

This website allows you to view and edit information stored on equivalent computer science courses offered by outside universities.
